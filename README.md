# SC1 GitHub Pages Showcase

This folder is a standalone static copy of the SC1 site for GitHub Pages.

## What to publish

Publish the contents of this folder:
- `index.html`
- `webshop.html`
- `style.css`
- `app.js`
- `sc1-config.js`
- `sc1-live-stream-copy.js`
- `sc1logo.jpg`
- `sc1header.mp4`

## How to host on GitHub Pages

1. Create a new GitHub repository.
2. Upload the contents of this folder to the repository root.
3. In GitHub, go to `Settings` -> `Pages`.
4. Set the source to your default branch and the root folder.
5. Save and wait for the Pages URL.

## Notes

- This is a frontend showcase only.
- It does not require the Fight Manager backend to display the main landing pages.
- Some links still point to local development URLs for the fight manager, which is fine for a demo/mockup stage.
- The live stream area uses the copy file in `sc1-live-stream-copy.js` and the config in `sc1-config.js`.
